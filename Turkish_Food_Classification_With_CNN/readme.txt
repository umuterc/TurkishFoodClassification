Due to ethical issues the data that has been used in this project cannot be shared.

To train model yourself you can use public datasets on the internet. 

Once you have data set either name the folder(that has images) as "Turkish_Food_404" or change the directory name in the "train.ipynb" file.

For os operations in dataset folder check the "os_operations.ipynb"

!!!BE CAREFULL WHEN YOU ARE USING "os_operations.ipynb" FUNCTIONS AND READ FUNCTION DESCRIPTION CAREFULLY!!!
